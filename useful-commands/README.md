# useful-commands

Some commands I want to remember for some reason.

## Linux

- `type python3`
- `eval $(ssh-agent)`, `ssh-add -t 1800` (30 minutes), `eval $(ssh-agent -k)`
- `tmux new-session -As main`
- `tmux setw -g mouse on`
- `git log --graph --oneline`

## Git Bash

- `export MSYS_NO_PATHCONV=1`
- `winpty rclone ncdu .`
